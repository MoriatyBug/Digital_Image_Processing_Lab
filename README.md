# Digital Image Processing Lab
中国科学技术大学软件学院高级图像处理实验。

本实验共分成三个部分：五个基础实验+三个拓展实验+一个综合实验。

实验源码有不完善的地方，学业繁忙就没有时间去改进，欢迎提出issue。

## 一、实验环境

- Opencv 3.0
- Visual Studio 2015

## 二、目录结构描述


├─images		    / /  实验用的图片和视频文件

├─resources		    / /  实验要求和参考书籍

├─src			    / /  实验源代码    

└─static			/ /  实验报告
	└─assets		/ /  Markdown图片
